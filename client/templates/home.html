Angular Demo

This site is a demo of some basic functionality of Angular.js.

The site is based on the Lynda.com course "Up and running with angularjs 1." However I chose to build the lessons inside of Foundation for Apps, instead of using the course files, so I could see how that works.

Each page demonstrates a simple idea. So you can dig into the code and see how it works.
